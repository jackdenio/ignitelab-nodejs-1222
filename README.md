##  Technologies used

-  NodeJS
-  NestJS
-  Prisma

## Requests

 - Create Notification
 - Cancel Notification
 - Mark notification as read
 - Mark notification as unread
 - Count Notifications
 - Get Notifications

## Runnig the project

Clone the project

```bash
  git clone https://github.com/jackdenio/ignitelab-nodejs-1222
```

Enter the project directory

```bash
  cd ignitelab-nodejs-1222
```

Install with dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```
